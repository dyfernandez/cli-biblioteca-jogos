# 🎮 Game Library CLI

A command-line application to manage my personal Steam game collection.

## About

This project allows you to add, list, edit, soft-delete and restore games
from a local SQLite database — all through a simple terminal interface.

## Technologies

- Python 3
- SQLite3
- Object-Oriented Programming

## How to Run

```bash
 Clone the repository
git clone https://github.com/dyfernandez/cli-biblioteca-jogos.git

# Navigate to the project folder
cd cli-biblioteca-jogos

# Create and activate virtual environment
python -m venv .venv
.venv\Scripts\activate  # Windows

# Run the application
python main.py
```
